# Product Glossary

| Term | Definition |
|---|---|
| Career asset | A reusable professional achievement, role, contribution, output, relationship event, or experience. |
| Evidence item | A discrete source-backed fact or passage supporting an asset, criterion, or application claim. |
| Document | A locally stored original, derived, or generated file with metadata and an AI handling policy. |
| Original document | An uploaded file preserved unchanged at application level. |
| Derived value | A tag, theme, summary, link, or assessment created by AI or a deterministic rule rather than asserted by the user. |
| Strategic goal | A user-adopted short-, medium-, or long-term career objective. |
| Target | A role, award, fellowship, recognition outcome, appointment, or trajectory the user may pursue. |
| Criterion | A user-defined weighted requirement for a target. |
| Opportunity | A time-bound or actionable possibility that may advance career goals. |
| Strategic value | User or AI assessment from 1–5 of how strongly an opportunity advances adopted goals. |
| Probability | Estimated likelihood from 0–100%; an input estimate rather than an objective forecast. |
| Effort | Estimated pursuit cost from 1–5, where 5 is highest. |
| Priority score | Deterministic 0–100 score derived from strategic value, probability, and effort. |
| Urgency | Deadline-related time pressure displayed separately from priority. |
| Readiness | Weighted coverage of the adopted criteria for a target. |
| Confidence | Strength of the evidence and assessment, not the probability of success. |
| Career trajectory | A proposed sequence of milestones from current state to a desired future state. |
| Adopted | Explicitly accepted by the user as part of their strategy or requirements. |
| Suggested | Proposed by AI but not authoritative or adopted. |
| Grounded claim | A generated factual statement internally linked to supporting evidence. |
| AI allowed | Document content may be selectively sent to the external AI provider. |
| Local only | Document content must not leave the local CAPSTONE system. |
| Redacted | Only a reviewed redacted derivative may be sent to the AI provider. |
| Public professional information | Information lawfully public in an official professional context and not sensitive or confidential. |
